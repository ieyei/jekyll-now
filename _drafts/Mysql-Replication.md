## MySQL Replication
데이터 정합성이 중요한 비즈니스에서 MySQL Replication은 데이터 불일치 이슈가 존재  
Performance => Asynchronous(데이터 정합성 문제)  
High Availability => Replication  

### Replication Types

* Asynchronous Replication
* Semi-Synchronous Replication
* Synchronous Replication
